Portfolio Cybersecurity April 2025

How I Set Up My Cybersecurity Portfolio

Initially I had an assignment to get my own personal domain, set up a branded email, create a GitHub portfolio page, and showcase things I have done. I really enjoyed doing it. I have known for a while that I should be taking these kinds of steps, but never really took the time and effort to set myself up for career success. Today however I can say that I have.

I registered my very own domain name and used it to create a personal email that I actually own. Not Gmail or Yahoo. It is me@amcclarin.com and that is mine. It feels special knowing that. I set up Cloudflare for DNS management and privacy protection on my domain, and used Cloudflare Email Routing to forward messages to an existing inbox so I can receive and eventually send emails from my own domain. I also created a GitHub repository and built a portfolio page hosted on GitHub Pages that I know will grow into something much bigger over time.

I want to start doing these small projects and documenting everything I learn along the way, for myself and for anyone else who wants to see that I actually know what I am talking about. It is an invaluable skill that shows people I am doing the things I say I am doing. It was a lot of fun configuring everything and seeing it all come together, and I am happy knowing I am building the foundation for a bright future.
